Web Furniture

Website ini masih termasuk tugas dari bootcamp di Harisenin.com Web ini merupakan website furniture yang menjual berbagai macam furniture, didalamnya ada menu Home, About, Product & Contact. Web ini dibuat dengan Tailwind CSS & React JS dengan halaman page yang terpisah menggunakan reactdom.
